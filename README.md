## Overview

This repository implements the **SIDARTHE** compartmental model proposed by
[Giordano *et al.* (2020)](https://www.nature.com/articles/s41591-020-0883-7)
to simulate the spread of COVID-19 in Italy under successive public-health
interventions.

The model tracks eight population compartments:

| Symbol | Meaning |
|--------|---------|
| **S** | Susceptible |
| **I** | Infected – asymptomatic, undetected |
| **D** | Diagnosed – asymptomatic, detected |
| **A** | Ailing – symptomatic, undetected |
| **R** | Recognized – symptomatic, detected |
| **T** | Threatened – acutely symptomatic, detected |
| **H** | Healed |
| **E** | Extinct |

The system of 8 coupled ODEs is solved numerically using a hand-written
**Runge-Kutta 4th order (RK4)** integrator (no `scipy.integrate` dependency).

## Model Equations

$$\frac{dS}{dt} = -S(\alpha I + \beta D + \gamma A + \delta R)$$

$$\frac{dI}{dt} = S(\alpha I + \beta D + \gamma A + \delta R) - (\epsilon+\zeta+\lambda)I$$

$$\frac{dD}{dt} = \epsilon I - (\eta+\rho)D \qquad \frac{dA}{dt} = \zeta I - (\theta+\mu+\kappa)A$$

$$\frac{dR}{dt} = \eta D + \theta A - (\nu+\xi)R \qquad \frac{dT}{dt} = \mu A + \nu R - (\sigma+\tau)T$$

$$\frac{dH}{dt} = \lambda I + \rho D + \kappa A + \xi R + \sigma T \qquad \frac{dE}{dt} = \tau T$$

### Basic Reproduction Number

$$R_0 = \frac{1}{r_1}\!\left[\alpha + \frac{\beta\epsilon}{r_2} + \frac{\gamma\zeta}{r_3} + \frac{\delta\eta\epsilon}{r_2 r_4} + \frac{\delta\zeta\theta}{r_3 r_4}\right]$$

where $r_1 = \epsilon+\zeta+\lambda$, $r_2 = \eta+\rho$, $r_3 = \theta+\mu+\kappa$, $r_4 = \nu+\xi$.

## Intervention Scenarios & R₀ Values

| Phase | Intervention | R₀ |
|-------|-------------|-----|
| Initial | No intervention | 2.38 |
| Day 4 | Social distancing | 1.66 |
| Day 12 | Complete lockdown | 1.80 |
| Day 22 | Partial lockdown | 1.60 |
| Day 28 | Full lockdown | **0.99** |
| Day 38 | Wider testing campaign | **0.85** |

R₀ < 1 signals the epidemic is under control.

## Results

Each plot shows the evolution of all eight compartments over 200 days.
Key observations:

- **Initial scenario** (R₀ = 2.38): rapid depletion of susceptibles, large
  infected peak, significant extinct fraction.
- **After Day 28 full lockdown** (R₀ = 0.99): the epidemic just barely
  controlled; most of the population remains susceptible.
- **After Day 38 wider testing** (R₀ = 0.85): epidemic collapses quickly;
  susceptible fraction stays near 1.0 throughout the simulation.

## Numerical Method

The RK4 update rule used:

$$y_{n+1} = y_n + \frac{h}{6}(k_1 + 2k_2 + 2k_3 + k_4) + \mathcal{O}(h^5)$$

A time grid of 2000 points over 365 days (step ≈ 0.18 days) gives
excellent accuracy for this stiff-ish system.
